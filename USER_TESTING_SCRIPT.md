# Fixaro User Testing Script

Use this script with 5 real mobile users before final launch approval.

## Test Instructions

Ask each user to do this on their own phone:

1. Open `https://fixarosolutions.com`.
2. Without scrolling much, tell us what this business does.
3. Find the AC service price.
4. Find what to do if the AC is not cooling.
5. Check if service is available in Banaswadi or Frazer Town.
6. Try to fill the booking form until WhatsApp opens.
7. Do not send the WhatsApp message. Just check whether it looks correct.
8. Tell us what confused you.
9. Tell us what made you trust or not trust the business.
10. Rate the website from 1 to 5 for trust, speed, ease of booking, mobile design, and clarity of pricing.

## Observer Notes

- Do not help unless the user is fully stuck.
- Note the exact step where the user hesitates.
- Check whether the user notices pricing, warranty, service area search, and Call/WhatsApp actions.
- Stop before any real message is sent in WhatsApp.
